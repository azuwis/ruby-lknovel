# Lknovel

Generate epub from http://lknovel.lightnovel.cn/

## Installation

Install using gem:

    $ gem install lknovel

Or add this line to your application's Gemfile:

    gem 'lknovel'

And then execute:

    $ bundle

## Usage

    $ lknovel http://lknovel.lightnovel.cn/main/book/2136.html
